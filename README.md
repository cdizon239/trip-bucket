## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/cdizon239/trip-bucket.git
   ```

2. install modules
   ```sh
   npm install
   ```

3. Run node
   ```sh
   node index.js
   ```
